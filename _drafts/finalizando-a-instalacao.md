Finalizando a instalação

*''Definir a senha do usuário root:''
```
# passwd
```
*''Mudar a raiz:''
```
# exit
```
*''Desmontar os sistemas de arquivos:''
```
# umount -R /mnt
```
*''Reiniciar o sistema:''
```
# reboot
```
